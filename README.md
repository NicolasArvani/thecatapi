## Aplicativo para ver fotos e informações de gatinhos!!!

### Usando a API do site https://thecatapi.com/

### Para rodar o projeto

```bash
flutter pub get
flutter run
```

### Para gerar o apk

```bash
flutter build apk --split-per-abi
```
